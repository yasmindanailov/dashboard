Estados

02

Hover y active se demuestran interactivamente — pasa el cursor / haz click sobre los botones de arriba. Lo de abajo son los estados fijos que no se ven sin interacción.

Disabled

opacity: 0.5 · cursor: not-allowed. Pierde hover. Para "deshabilitado accesible" usar aria-disabled + handler que cancele la acción.

Loading

Contenido oculto, spinner centrado, pointer-events: none + aria-busy="true".

Focus visible · ring doble

↑ visualización · navega con Tab sobre los botones de arriba para ver el comportamiento real.
Anillo doble: 2px del color de superficie + 2px de brand. Se mantiene legible sobre cualquier fondo. Definido en --focus-ring.

Focus sobre fondo oscuro

Para BulkActionBar y Tooltip, usar --focus-ring-on-dark (anillo claro sobre dark).
