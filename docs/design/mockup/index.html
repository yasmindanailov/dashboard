Aelium · Mockup vivo

Tu socio digital, a tu lado.

Maqueta del dashboard del programa de diseño. Crece fase a fase. Hoy contiene los primeros componentes (fase 2.A) y la firma visual que los hace reconocibles como Aelium — no como otro dashboard genérico. Todo alineado con el documento de marca v1.6.

Cómo navegar

01

Usa la barra lateral. Los enlaces marcados como pendientes (·) llevan a una página placeholder amable hasta que la fase correspondiente los entregue. Las páginas activas se actualizan a medida que el programa avanza.

Fase 2 · Componentes base Button, Input, Select, …
Cada componente tiene su propia página con todas las variantes, tamaños y estados. Anotaciones a la derecha. Tokens consumidos referenciados explícitamente.
Fase 3 · Patrones ListPage 4v · DetailPage 3v · FormPage 3v
Wrappers responsables del 95% de las páginas. Variantes nativas (DD-029) por contexto: tabla / grid / timeline / split en listings; standard / with-aside / workspace-lite en detalles; standard / wizard / long-form en formularios. Ancho único 1200/1400, sin accent-stripe en recuadros (DD-030).
Fase 4 · Shells Auth · Cliente · Admin · Partner
Sidebar + topbar + content area por tipo de portal. Densidad por dialecto (cliente respira, admin denso). Acento por portal se decide aquí.
Fases 5–9 · Páginas Mockups completos
Cada pantalla del producto en su contexto real (overview, detalle, formulario, auth flow).

Estado actual

Fase 2.A · formularios — listos para revisión. Los 6 componentes están en la nav (Button, Input, Select, Textarea, SearchInput, Dropdown). Una página de muestra los compone en un caso real: Abrir un ticket de soporte.

Empieza por Firma visual para entender qué hace que un componente Aelium se sienta diferente de otros dashboards: símbolo de marca, loader con rombos, mesh sutil, numerales tabulares, accent stripe, card como acción.

Convenciones de la maqueta. Sin librerías externas. Cero hex hardcoded. Todo via tokens (tokens.css). Estilos compartidos en styles.css. Comportamiento compartido en scripts.js.
