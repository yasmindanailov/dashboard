Tu socio digital, a tu lado.

Maqueta del dashboard del programa de diseño. Crece fase a fase. Hoy contiene los primeros componentes (fase 2). A medida que avance, se irán incorporando patrones de página, shells por portal y mockups completos de las pantallas reales del producto. Todo lo que ves aquí está alineado con el documento de marca v1.6.

Cómo navegar

01

Usa la barra lateral. Los enlaces marcados como pendientes (·) llevan a una página placeholder amable hasta que la fase correspondiente los entregue. Las páginas activas se actualizan a medida que el programa avanza.

Fase 2 · Componentes base Button, Input, Select, …
Cada componente tiene su propia página con todas las variantes, tamaños y estados. Anotaciones a la derecha. Tokens consumidos referenciados explícitamente.
Fase 3 · Patrones List · Detail · Form
Plantillas reutilizables que componen los componentes en estructuras de página repetibles.
Fase 4 · Shells Auth · Cliente · Admin · Partner
Sidebar + topbar + content area por tipo de portal. Densidad por dialecto (cliente respira, admin denso). Acento por portal se decide aquí.
Fases 5–9 · Páginas Mockups completos
Cada pantalla del producto en su contexto real (overview, detalle, formulario, auth flow).
